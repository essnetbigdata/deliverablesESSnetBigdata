# deliverablesESSnetBigdata
Deliverables of ESSnet Big Data

This repository contains all deliverables from the ESSnet Big Data project, organized by Work Package (WP). The files follow the naming convention SGAa_p_d.pdf where:
- a = SGA number (1 or 2)
- p = Work Package number
- d = Deliverable number
